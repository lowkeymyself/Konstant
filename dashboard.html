UPGRADING Konstant is becoming a universal movement script. Coming soon.

Where this is going

Konstant started as a focused toolkit for one game — Roblox Supermoto. Through testing, the underlying physics, mod logic, and customization layer turned out to work across nearly every bike, e-bike, and scooter game on Roblox. So instead of staying narrow, we're widening the scope.

The next version detects the place ID at startup. In the original Supermoto place, it loads the full suite — economy exploits, the curated bike spawner, custom map tools. In any other game, it strips back to the universal core: physics tuning, ESP, anti-fall, optimizer, customization, and a generic spawner that hunts for bike folders and spawn remotes automatically. New brand name in that mode: Konstant Aero.

This is a structural change, not a rewrite. Everything you already use stays in the Supermoto build. The site, the script, and the docs are all moving together.
